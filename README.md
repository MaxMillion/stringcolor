## Demo

A GitHub-hosted demo can be found here:

## Install

Manual install:  
  1. Download [stringcolor.js](https://raw.githubusercontent.com/erming/stringcolor/gh-pages/stringcolor.js)
  2. Include source: `<script src="stringcolor.js"></script>`
